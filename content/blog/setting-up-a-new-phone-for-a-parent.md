Buying a parent a new phone is a wonderful gesture. But handing it over still in the box can turn that gift into weeks of "How do I…?" phone calls. With a little planning, you can set it up so it feels friendly and familiar from the very first tap.

## Start with the basics, done properly

Before anything else, take care of the foundations:

- **Sign in to the right account.** An Apple ID (iPhone) or Google account (Android) is the key to everything — apps, photos, and backups. Set this up carefully and write the password down somewhere safe.
- **Connect to home Wi-Fi.** This keeps things fast and avoids using up mobile data.
- **Transfer their old phone.** Both Apple and Android have built-in tools to copy contacts, photos, and messages across. Nothing important should be left behind.

## Make it easy to see and use

This is where a phone goes from "usable" to "genuinely comfortable":

- Turn up the **text size** and **display brightness**.
- Put the **apps they actually use** — phone, messages, camera, WhatsApp — on the first screen, and tuck the rest away.
- Set up **favorite contacts** so calling family is one tap.
- Consider a simple, protective **case** and a **screen protector**.

## Set up the apps that matter

Ask which apps they want, then install and sign in to each one *with* them, so nothing is a mystery later. Common favorites: WhatsApp, a photos app, their bank, and a video-calling app to see the grandkids.

## Protect them from scams

Take five minutes to explain the golden rule: **no one legitimate will ever ask for a password, a gift card, or remote access over the phone.** Turning on spam-call filtering helps too.

## Don't rush the handover

The setup matters, but so does the first lesson. Sit together, make one video call, send one photo, and answer questions patiently. Confidence is what makes the phone stick.
